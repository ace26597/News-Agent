HOW TO USE
================================================================================

STEP 1: Install dependencies
  > pip install -r requirements.txt

STEP 2: Configure API keys in constants.py
  (Already configured with your keys)

STEP 3: Integrate into your Flask app
  from ome_blueprint import ome_blueprint
  app.register_blueprint(ome_blueprint, url_prefix='/OME')

STEP 4: Access at /OME/ endpoint
  http://your-domain/OME/
